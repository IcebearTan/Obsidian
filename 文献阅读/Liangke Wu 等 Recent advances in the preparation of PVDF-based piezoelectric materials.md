# 文章整体概述

这是一篇 2022 年发表在《Nanotechnology Reviews》上的综述，标题为：

**Recent advances in the preparation of PVDF-based piezoelectric materials**

作者：Liangke Wu 等（重庆大学）

**核心主题：** 系统总结 PVDF 及其共聚物压电材料的**晶体结构、表征、相转变、制备方法、增强策略、机理、典型参数与对比**，重点围绕如何提高压电 β 相含量、结晶度、链取向、偶极排列，以弥补 PVDF 压电系数低于 PZT 的短板。

**研究意义：** PVDF 柔性、轻质、环保、耐酸碱、生物相容，适合柔性能量收集 / 传感，但压电性能弱（20–40 pC/N，比 PZT 低一个数量级），亟需高效制备策略。

---

# 一、引言（研究背景与关键痛点）

1. **压电材料对比**

- PZT：压电常数高、机电耦合强，但**脆性、昂贵、密度大、含铅有毒**，限制柔性 / 可穿戴应用。
- PVDF 及其共聚物：**柔性好、环保、轻质、耐卤 / 酸、生物相容、可成曲面**，但**压电性弱**，成为应用瓶颈。

2. **压电本质**

- 基于**压电效应（发电）**与**逆压电效应（驱动）**。
- PVDF 是半结晶聚合物，具有压电 / 热释电 / 铁电性。

3. **压电性能的四大决定因素（全文核心）**

- **β 相含量**：β 相压电最强；α 相最稳定、压电最弱。
- **结晶度**：仅结晶区有压电性，无定形区无贡献。
- **分子链取向**：无规链→输出极弱；高度取向→强压电。
- **偶极矩排列**：无序→相互抵消、无压电；定向排列→强压电。

4. **现有问题**

- α→β 相转变机理**不清晰**。
- 同类实验常出现**矛盾结果**。
- 亟需系统总结制备与增强策略。

# 二、PVDF 及其共聚物的基本性质

## 2.1 PVDF 晶体结构（多晶型）

PVDF 至少 5 种晶相：**α、β、γ、δ、ε**

- **α 相（最常见、最稳定）**：单斜晶，TGTG' 构象，**压电惰性**。
- **β 相（最关键、压电最强）**：正交晶，**全反式 TTTT**构象，**压电活性**。
- **γ 相**：正交晶，T3GT3G，弱压电。
- **δ 相**：α 相的极性变体。
- **ε 相**：δ 相的反极性变体。

**核心目标：提高 β 相比例、促进 α→β 转变。**

## 2.2 晶体结构表征方法

### （1）XRD（X 射线衍射）

- α 相特征峰：2θ=17.7°(100)、18.5°(020)、19.9°(110)、26.5°(021)
- β 相特征峰：2θ=20.6°(110)、36.3°(200)
- **结晶度计算**：Xc = Ic/(Ia+Ic)（Ic：结晶峰面积；Ia：无定形峰面积）

### （2）FTIR（傅里叶红外）

- α 相：765 cm⁻¹、615 cm⁻¹、1170 cm⁻¹
- β 相：840 cm⁻¹、510 cm⁻¹、1280 cm⁻¹
- γ 相：1220 cm⁻¹
- **β 相相对含量**：F (β) = Aβ/(1.26Aα + Aβ)（Aα：765 cm⁻¹ 吸光度；Aβ：840 cm⁻¹ 吸光度）

### （3）DSC（差示扫描量热）

- 结晶度：Xc = (ΔHm/ΔHm°)×100%
- ΔHm：样品熔融焓；ΔHm°：100% 结晶 PVDF 熔融焓（104.5 J/g）

## 2.3 相转变规律

- **α 相最稳定**，需外界能量（拉伸、极化、热处理、填料）才能转为 β 相。
- 核心路径：**α → β**；拉伸 + 极化是最有效手段。

## 2.4 分子链模拟（理论机理）

- **α→β 能垒：16.16 kJ/mol**；反向仅 6.25 kJ/mol（β 易转回 α）。
- 纳米填料（如 CNT）**降低能垒、稳定 β 相**：
    
    - β 链优先吸附在 CNT 表面；
    - 远离 CNT 的 α 链先转 β，再吸附到 CNT。
    
- **电场极化机理**：电场使链拉伸 / 压缩、偶极定向；>100 MV/cm 可直接 α→β（实验仅需 < 100 MV/m 即可）。
- **缺陷影响**：间隙缺陷不稳定；空位影响 γ 相稳定性。
- **纤维尺寸效应**：超细纤维中 β 相易转顺电相。

## 2.5 主要共聚物

### （1）PVDF-HFP（六氟丙烯共聚物）

- 特点：**柔性↑、疏水性↑、结晶度↓**（CF₃大位阻）。
- 性能：d₃₁≈PVDF，热释电系数更高。

### （2）PVDF-TrFE（三氟乙烯共聚物）

- 特点：**易形成 β 相、无需拉伸即可极化、结晶度高、居里温度 Tc 低于熔点 Tm**。
- 结构：β 相峰 2θ=19.9°、FTIR=850 cm⁻¹；晶粒呈棒状。
- 优势：机电耦合强、可低温极化。

# 三、PVDF 基压电复合材料的制备方法

## 3.1 初始结晶（成膜 / 成型）

### （1）溶液浇铸（Solution Casting，最常用）

**步骤：**

1. 溶解：PVDF 溶于 DMF/DMAc/DMSO/ 丙酮等；超声（5–180 min）+ 搅拌（>30 min）加速溶解。
2. 成膜：浇铸于玻璃 / 铝基底；**50–120℃蒸发 2 h–3 天**。
    
    **关键：高温易生成 α 相；低温利于 β 相。**

### （2）热压（Hot Press）

- 工艺：150–200℃、5–30 MPa、5–60 min；熔融→压片→冷却。
- 优势：**无溶剂、致密度高、消除孔隙**；常与静电纺丝复合。

### （3）静电纺丝（Electrospinning，制备超细纤维首选）

- 优势：**原位拉伸 + 原位极化**，无需后处理即可得高 β 相（最高 93%）。
- 机理：射流拉伸使链沿纤维轴取向、β 相成核；电场直接极化偶极。
- 关键参数：
    
    - 电压：20 kV 最优；
    - 接收辊转速：9000 rpm、100℃时 β 相最高；
    - 分子量：高分子量→高粘度→充分拉伸→高 β 相。
    

## 3.2 β 相形成与 α→β 转变的增强策略（全文重点）

### （1）添加纳米填料（最主流）

**作用：** 异相成核、降低能垒、诱导 β 相、提高结晶度、降低极化场。

**常用填料与效果：**

- **碳纳米管 CNT**：0.2–1 wt% 最优；β 相至 96%、d₃₃=23 pC/N。
- **石墨烯 / GO/rGO**：<0.1 wt% 即可；β 相至 97%、电压翻倍。
- **碳黑 CB**：0.5 wt%；电压翻倍。
- **PZT**：30 wt%；d₃₃=84 pC/N（纯 PVDF=32）。
- **金属纳米颗粒（Au/Ag）**：诱导 β 相、形成球晶。
- **铁氧体（CoFe₂O₄）**：磁场下 β 相至 95%。

**混合填料（协同）：** rGO/TiO₂、MnO₂/ 石墨烯 / CNT、BT/GNS 等，**极化场降至 50–80 MV/m**。

**注意：填料过量→团聚→结晶度↓、缺陷↑、压电性↓。**

### （2）机械拉伸（最有效相变手段）

**作用：** 应力诱导 α→β、链取向、偶极定向、结晶度↑。

**关键参数：**

- 温度：50–140℃，**80℃最优**。
- 拉伸比：**>4（最优 5–6）**，>8 易开裂。
- 速率：1–3 mm/min。

**相变过程：** α 球晶→应力破碎→重排成 β 原纤→偶极垂直拉伸方向排列。

### （3）热处理 / 退火

**作用：** 溶剂去除、链重排、β 相结晶度↑。

**关键参数：**

- 温度：**60–140℃（<100℃最优）**；>140℃→无定形↑、β 相↓、γ 相生成。
- 时间：10 min–48 h。
- 特殊：**紫外退火（350 μs）**→Pr=5.4 μC/cm²、矫顽场 120 MV/m。

### （4）极化（Poling，激活压电性）

**作用：** 偶极定向排列、消除无序抵消、增强 α→β。

**方式：**

- 接触极化：80–100℃、50–160 MV/m、30–60 min。
- 电晕极化：无接触、适合大面积、3D 打印原位极化。
- 原位极化：静电纺丝 / 3D 打印时同步完成。

**关键：** 场强 < 160 MV/m（防击穿）；超薄膜（1 μm）可承受 2 GV/m。

### （5）磁场 / 电场诱导填料取向

- **磁场（CoFe₂O₄）**：磁致伸缩→界面应力→β 相至 95%。
- **电场**：对齐碳纳米纤维→链取向、压电增强。

# 四、方法对比与综合效果

## 4.1 各增强方法优缺点（核心总结）

- **纳米填料**：优点：β 相↑、极化场↓；缺点：过量团聚、结晶度↓。
- **拉伸**：优点：相变最强、链 / 偶极定向；缺点：易开裂、结晶度可能↓。
- **热处理**：优点：结晶度↑、β 相稳定；缺点：高温易转 α/γ。
- **极化**：优点：偶极定向、压电激活；缺点：高场易击穿。
- **填料取向**：优点：界面应力、协同增强；缺点：设备复杂、数据少。

## 4.2 典型案例数据（表 4 核心提取）

- PVDF+CNT（0.2 wt%）：拉伸 + 极化→β=96%、d₃₃=23 pC/N。
- PVDF+rGO（0.1 wt%）：退火→β>95%、d₃₃=39.3 pC/N。
- PVDF+PZT（30 wt%）：退火→d₃₃=84 pC/N。
- PVDF 静电纺丝 + 退火：β=87.4%。
- PVDF+CoFe₂O₄+ 磁场：β=95%。

# 五、结论

1. PVDF 及其共聚物是柔性压电的核心材料，**β 相、结晶度、链取向、偶极排列**是压电性能四大关键。
2. 主流制备：**溶液浇铸、热压、静电纺丝**；静电纺丝可原位拉伸极化，效率最高。
3. 增强策略：**纳米填料、拉伸、热处理、极化、填料取向**；**拉伸 + 极化 + 少量填料**组合效果最优。
4. 现存挑战：**相转变机理不清、参数交互复杂、实验结果矛盾**，需多尺度模拟与实验结合进一步研究。
